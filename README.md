# Eurosoft MS Géolocalisation

## Prérequis
Assurez-vous d'avoir [Node.js](https://nodejs.org/) installé sur votre ordinateur.

## Comment lancer le projet (How to run)

1. **Ouvrir le projet :**
   Ouvrez un terminal (ou l'invite de commande) et naviguez jusqu'au dossier du projet téléchargé depuis GitHub.

2. **Installer les dépendances :**
   Tapez la commande suivante pour télécharger toutes les librairies nécessaires :
   ```bash
   npm install
   ```

3. **Lancer le serveur local :**
   Une fois l'installation terminée, démarrez le projet avec :
   ```bash
   npm run dev
   ```

4. **Voir le résultat :**
   Le terminal vous donnera un lien local (généralement `http://localhost:5173/` ou `http://localhost:3000/`). Cliquez dessus (ou copiez-le dans votre navigateur) pour voir le site !
